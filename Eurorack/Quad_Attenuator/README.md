# Quad Attenuator

A four-channel attenuator Eurorack module.

## What it does

Provides four independent attenuation channels for audio or CV signals.

## Files

This directory contains panel design files only:

| File                                    | Format    | Description                                |
| --------------------------------------- | --------- | ------------------------------------------ |
| `Quad_Attenuator_Panel.svg`             | SVG       | Final panel artwork                        |
| `Quad_Attenuator_Panel_PATHS.svg`       | SVG       | Panel with path outlines for laser cutting |
| `Quad_Attenuator_Panel.lbrn2`           | LightBurn | Laser cutter project file                  |
| `Quad_Attenuator_Panel_OldVersion.svg`  | SVG       | Earlier panel revision                     |
| `Quad_Attenuator_Panel_OldVersion2.svg` | SVG       | Earlier panel revision 2                   |
