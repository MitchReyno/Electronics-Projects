# Dual Log/Linear VCA

A dual voltage-controlled amplifier (VCA) Eurorack module with switchable logarithmic and linear CV response curves.

## What it does

Provides two independent VCA channels, each with:

- Audio input and output jacks
- CV input for gain control
- Switchable **logarithmic** (musical) or **linear** gain response

## Files

KiCad 8 project files are in the root directory (`.kicad_sch`, `.kicad_pcb`, `.kicad_pro`).
