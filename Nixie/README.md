# Nixie

High-voltage power supply projects for Nixie and Dekatron display tubes.

## Projects

| Project | Description |
|---------|-------------|
| [Nixie PSU MK1](Nixie_PSU_MK1/) | Boost converter providing ~170 V DC for Nixie tube displays |
| [Dekatron PSU MK1](Dekatron_PSU_MK1/) | High-voltage boost converter adapted for Dekatron counting tubes |

Both PSUs are built around the **MC34063AP** switching regulator IC with an **IRF740** N-channel MOSFET as the power switch.
